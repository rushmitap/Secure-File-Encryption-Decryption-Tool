This project is a simple and secure file encryption and decryption tool built using the AES (Advanced Encryption Standard) algorithm. 
It allows users to encrypt any file using a secret key and later decrypt it back to its original form using the same key.

Features:
- Encrypt files using AES algorithm
- Decrypt files with the correct key
- Supports multiple file types (text, images, documents, etc.)
- Key-based security system
- Fast and efficient processing
- Simple and user-friendly interface
